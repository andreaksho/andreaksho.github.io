Day 35 - Set Your Settings

I really liked creating this UI today, because I got to create everything within it.

I copied the Display Settings page that LGG5 has. The most fun about today were the logos. I got to create every one of them, except for the home icon, but everything else, I created. Everyday, I feel I'm getting more and more confident using the shapes within Sketch.

As for the presentation, I wanted to do something creative by adding other pages beside it at an angle with a shadow, similar to my Calculator UI. . But because I only created one page, I was only create that effect by duplicating the page. Better than nothing I guess? I just wanted to try the effect out. :)
